# Portfolio — page-web

Este repositório contém um pequeno site estático (HTML/CSS/JS) que funciona como um portfolio simples.

Visão geral
- Site: arquivo `index.html` com estilos em `style.css` e scripts em `script.js`.
- Objetivo: demonstrar projetos e trabalhar como página pessoal/portfolio.

Como visualizar localmente
1. Abra `index.html` no navegador (duplo clique ou `File → Open`).
2. Ou sirva localmente com um servidor estático, por exemplo:

```
python3 -m http.server 8000
```

Deploy (GitHub Pages)
- Habilitado automaticamente através de um workflow GitHub Actions que publica o conteúdo da branch `main` para a branch `gh-pages`.
- Após um push para `main`, a página será publicada em `https://<usuario>.github.io/<repo>` (ou em um domínio personalizado se configurado).
